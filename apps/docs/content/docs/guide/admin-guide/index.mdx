## Admin Privileges

Administrators can:
- Create, edit, and deactivate employees
- Manage organization members, invitations, and manager assignments
- Assign multiple managers to employees
- Create and manage teams and projects
- Configure surcharge rules for premium pay
- Grant granular permissions to employees
- Override vacation allowances
- View all employee information
- Access system audit logs and project reports
- Use the top-level Compliance overview to monitor audit evidence, workforce policy risk, and sensitive control changes
- Configure webhooks and API keys
- Manage organization OAuth providers
- Control app access permissions
- Configure payroll exports for file formats and supported API connectors
- Monitor payroll, audit, and scheduled export activity from the export operations cockpit
