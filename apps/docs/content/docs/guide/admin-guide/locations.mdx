## Understanding Locations

Locations represent physical offices, branches, or work sites within your organization. They are used for:
- Associating employees with work locations
- Location-specific holiday calendars
- Reporting and analytics by location
- Time zone management

---

## Creating Locations

To add a new location:

1. Go to **Settings** → **Organization** → **Locations**
2. Click **"Add Location"**
3. Enter location details:
   - **Name**: Location identifier (e.g., "Berlin Office", "Munich Branch")
   - **Address**: Physical address
   - **Time Zone**: Local time zone for the location
   - **Country**: Country for holiday calendar assignment
4. Click **"Create"**

---

## Managing Locations

**Edit Location:**
1. Find the location in the list
2. Click **"Edit"**
3. Update the information
4. Click **"Save"**

**Assign Employees to Location:**
1. Go to the employee's profile
2. Select the location from the **"Work Location"** dropdown
3. Save changes

---

## Location-Based Features

- **Holiday Calendars**: Each location can have country-specific holidays
- **Time Zone Awareness**: Time entries respect local time zones
- **Location Reports**: Filter analytics by location
- **Team Distribution**: See where team members are located
