## Understanding Locations

Locations represent physical offices, branches, or work sites within your organization. They are used for:
- Associating employees with work locations
- Location-specific holiday calendars
- Reporting and analytics by location
- Time zone management

---

## Creating Locations

To add a new location:

1. Go to **Settings** → **Organization** → **Locations**
2. Click **"Add Location"**
3. Enter location details:
   - **Name**: Location identifier (e.g., "Berlin Office", "Munich Branch")
   - **Address**: Physical address
   - **Time Zone**: Local time zone for the location
   - **Country**: Country for holiday calendar assignment
4. Click **"Create"**

---

## Managing Locations

**Edit Location:**
1. Find the location in the list
2. Click **"Edit"**
3. Update the information
4. Click **"Save"**

**Assign Employees to Location:**
1. Go to the employee's profile
2. Select the location from the **"Work Location"** dropdown
3. Save changes

---

## Location-Based Features

- **Holiday Calendars**: Each location can have country-specific holidays
- **Time Zone Awareness**: Time entries respect local time zones
- **Location Reports**: Filter analytics by location
- **Team Distribution**: See where team members are located

---

## Understanding Subareas

Subareas allow you to divide locations into smaller organizational units. This is useful for:

- Large offices with multiple floors or departments
- Campus environments with multiple buildings
- Warehouses with different zones or sections
- Any location that needs granular organization

![Location Detail with Subareas](/images/admin-guide/location-detail-subareas.png)

---

## Creating and Managing Subareas

### Creating a Subarea

![Subarea Dialog](/images/admin-guide/subarea-dialog.png)

1. Go to **Settings** → **Organization** → **Locations**
2. Click on the location you want to add subareas to
3. In the location detail view, find the **Subareas** section
4. Click **"Add Subarea"**
5. Enter the subarea details:
   - **Name**: Descriptive name (e.g., "3rd Floor", "Warehouse Zone A")
   - **Code**: Optional short identifier
   - **Description**: Additional context about the subarea
6. Click **"Create"**

### Editing a Subarea

1. Navigate to the location's detail page
2. Find the subarea in the list
3. Click the **edit icon** next to the subarea
4. Update the information
5. Click **"Save"**

### Deleting a Subarea

1. Navigate to the location's detail page
2. Find the subarea in the list
3. Click the **delete icon**
4. Confirm the deletion

**Note:** Deleting a subarea will remove it from all employee assignments. Consider reassigning employees first.

---

## Assigning Employees to Subareas

Employees can be assigned to specific subareas within their location:

1. Go to the employee's profile
2. First, ensure a **Work Location** is selected
3. If the location has subareas, a **Subarea** dropdown will appear
4. Select the appropriate subarea
5. Save changes

### Reporting by Subarea

Once employees are assigned to subareas:
- Filter time reports by subarea
- View team distribution across subareas
- Track attendance patterns by area
