Get up and running with the z8 Timer desktop app in just a few minutes.

## Quick Setup

<Steps>
  <Step>
    **Download the app** for your operating system

    - Windows: `.msi` or `.exe` installer
    - macOS: `.dmg` disk image
    - Linux: `.deb` package or `.AppImage`

    [View installation instructions →](/docs/desktop/getting-started/installation)
  </Step>
  <Step>
    **Configure your server URL**

    On first launch, enter your organization's Z8 webapp URL (e.g., `https://time.yourcompany.com`)
  </Step>
  <Step>
    **Sign in with your Z8 account**

    Click "Sign in with Z8" - your browser opens for secure authentication

    [View authentication details →](/docs/desktop/getting-started/authentication)
  </Step>
  <Step>
    **Start tracking time**

    Click the clock button to clock in - that's it!

    [View time tracking guide →](/docs/desktop/features/time-tracking)
  </Step>
</Steps>

## System Requirements

### Windows

- Windows 10 or later
- WebView2 Runtime (included in Windows 11, auto-installed on Windows 10)
- 50 MB disk space

### macOS

- macOS 10.15 (Catalina) or later
- Apple Silicon (M1/M2/M3) or Intel processor
- 50 MB disk space

### Linux

- WebKitGTK 4.1 or later
- GTK 3
- 50 MB disk space

---

## What You'll Need

Before installing the desktop app, ensure you have:

1. **Z8 Account** - An active user account in your organization's Z8 instance
2. **Webapp URL** - Your organization's Z8 server URL (ask your administrator)
3. **Internet Connection** - For initial setup and authentication (offline mode works after)
