## Platform Support

| Platform | Installer | Notes |
|----------|-----------|-------|
| **Windows** | `.msi`, `.exe` | Supports Windows 10 and newer systems, with WebView2 required for the app shell. |
| **macOS** | `.dmg` | Supports macOS 10.15+ on Intel and Apple Silicon devices. |
| **Linux** | `.deb`, `.AppImage` | Requires WebKitGTK 4.1+ and common GTK desktop dependencies. |
