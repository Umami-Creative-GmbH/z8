## Accessing the Calendar

The calendar provides a visual overview of your work schedule, absences, and team events:

1. Go to **Calendar** from the main navigation
2. Choose your view: **Day**, **Week**, or **Month**
3. Navigate using the arrow buttons or date picker

---

## Calendar Features

- **Your Work Periods**: See when you clocked in/out
- **Approved Absences**: Vacation days, sick leave shown in different colors
- **Team Events**: See when teammates are off (if permitted)
- **Holidays**: Public holidays highlighted

---

## Managing Work Periods

You can manage your work periods directly from the calendar:

### Edit a Work Period

1. Click on a work period in the calendar
2. Select **"Edit"** from the context menu
3. Adjust start time, end time, or notes
4. Click **"Save Changes"**

### Split a Work Period

If you need to add a break retroactively:
1. Click on the work period
2. Select **"Split"**
3. Enter the break start and end times
4. The period will be split into two segments

### Delete a Work Period

1. Click on the work period
2. Select **"Delete"**
3. Confirm deletion (note: may require approval for past entries)

---

## Filtering the Calendar

Use the filter options to customize your view:
- Show/hide different event types
- Filter by team members
- Toggle between personal and team calendar
